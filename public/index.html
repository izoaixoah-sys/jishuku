🔊 系统语音
推荐：使用 Microsoft Edge 浏览器
Edge 已内置七海（Nanami）、圭太（Keita）等 Neural 高品质日语语音，无需额外安装。

如果在 Edge 中仍看不到这些语音，可手动下载：
Edge 设置 → 语言 → 日本語 → 管理语音 → 下载 Nanami / Keita

安装后刷新本页面，语音列表会自动更新。
